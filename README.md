# 穴場紹介のレビューサイト

## サイト概要
### サイトテーマ
地域の穴場スポットを投稿、検索できるレビューサイト
​
### テーマを選んだ理由
私が暮らしていた地域には、シャッター通りという呼び名が定着しているほどがらんとした商店街があります。日中は人通りも少ないのですが、通勤時に歩いていると新しいお店が出店されていたり、趣のある着物屋や小物屋があったり、雰囲気のいい喫茶店があったりと意外と知られていないけれど素敵なお店がひっそりとあります。
また、私が勤めていた職場の運営団体が地域活性化に取り組んでいたこともあり、「こうした場所をどうすれば多くの人に知ってもらえるか？」ということを考えるようになりました。その結果、地域の穴場スポットを紹介・共有できるサイトがあったらいいなと思い、このテーマを選びました。
​
### ターゲットユーザ
* 来店する客を増やしたい店主
* 良かったと感じたお店を広めたり、誰かと共有したい人
* 穴場を探している住民や観光客
​
### 主な利用シーン
* 通りかかったときに見つけたお店を記録・共有したいとき
* 投稿されたお店の情報を検索・共感したり、コメントを送って反応したいとき
